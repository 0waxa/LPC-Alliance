## 使用

1. 下载Krzyk smoka Launcher（下载地址：[点我下载](https://github.com/0waxa/PCL-XAML/releases/download/XAML/Krzyk.smoka.Launcher.Pro.Max.zip)）
2. 打开Krzyk smoka Launcher 等待一下（配置文件存储在Github 下载较慢）
3. 主页成功加载即可使用
